Cread una aplicación o pagina web usando inteligencia artificial que permita iniciar sesión a usuarios usando una base de datos, dicha aplicación o pagina web deberá pedir usuario y contraseña además de reconocimiento facial para autenticar que la persona iniciando sesión es la misma que creo la cuenta.

La aplicación o pagina web solo deberá mostrar un login además de un mensaje si el usuario ha accedido correctamente o no, no hace falta que el resto tenga contenido, solo que el sistema de login con reconocimiento facial funcione, además de una forma para crear cuentas.

Podéis usar API's externas o usar una IA local para el sistema de reconocimiento facial.

Se entregara los archivos .cs o el contenido de la pagina web según el tipo de proyecto elegido, además de la base de datos (o un diagrama de su estructura).

[Importante]: Si usáis una API, no entreguéis las Keys
el stacktecnologico sera: Supabase, nextjs, typscript, vercel,react, shadcn para diseño....
